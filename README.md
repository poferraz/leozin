# Leonardo Veiga Photography Portfolio

A modern, responsive portfolio website showcasing photography and filmmaking work.

## 🚀 Quick Deploy to Netlify

1. **Zip the entire project folder** (or drag the folder directly)
2. **Go to [netlify.com](https://netlify.com)**
3. **Drag and drop** the folder/zip onto the deploy area
4. **Your site will be live** with a random URL (you can customize it later)

## 📁 Project Structure

```
leozin/
├── index.html              # Main portfolio page
├── netlify.toml           # Netlify configuration
├── _redirects             # URL redirects
├── robots.txt             # SEO configuration
├── sitemap.xml            # Search engine sitemap
└── assets/
    └── PORTFOLIO/
        ├── profile.png
        ├── profile-contact.jpg
        └── optimized/
            ├── images/     # Optimized photos
            ├── videos/     # Optimized videos  
            └── thumbnails/ # Video thumbnails
```

## ✨ Features

- **Responsive Design** - Works perfectly on all devices
- **Photo Galleries** - Interactive lightbox galleries by category
- **Video Carousel** - Modern carousel with video player modal
- **Performance Optimized** - WebP images, optimized videos, lazy loading
- **SEO Ready** - Meta tags, sitemap, robots.txt
- **Professional Animations** - Smooth scroll animations throughout

## 🎯 Sections

- **Hero** - Introduction and main title
- **Photos** - Gallery collections (Nature, Crows, Mushrooms, iPhone, Analog)
- **Videos** - Video carousel with high-quality playback
- **About** - Professional background and expertise
- **Contact** - Get in touch with profile image

## 🔧 Configuration

After deploying to Netlify:

1. **Custom Domain**: Add your own domain in Netlify settings
2. **Update URLs**: Replace `your-domain.netlify.app` in `sitemap.xml` and `robots.txt`
3. **Analytics**: Add Google Analytics or Netlify Analytics
4. **Forms**: Enable Netlify Forms if adding contact forms

## 📱 Browser Support

- Chrome (recommended)
- Firefox
- Safari
- Edge
- Mobile browsers (iOS Safari, Chrome Mobile)

## 🎨 Customization

The portfolio is built with vanilla HTML, CSS, and JavaScript for easy customization:

- **Colors**: Edit CSS variables in the `<style>` section
- **Content**: Update text directly in `index.html`
- **Images**: Replace files in `assets/PORTFOLIO/optimized/`
- **Videos**: Replace files in `assets/PORTFOLIO/optimized/videos/`
